Trace

The execution of a Cairo program produces a memory vector V and a matrix M of size L×3 with the evolution of the three registers pc, ap, fp. All of them with entries in F.

Construction of execution trace T:

In this section we describe the construction of the execution trace T. This is the matrix mentioned here in the description of the STARK protocol

  1. Augment each row of M with information about the pointed instruction as follows: For each entry (pci​,api​,fpi​) of M, unpack the pci​-th value of V. The result is a new matrix M∈FL×33 with the following layout
 A.  flags     (16) : Decoded instruction flags
 B.  res       (1)  : Res value
 C.  pointers  (2)  : Temporary memory pointers (ap and fp)
 D.  mem_a     (4)  : Memory addresses (pc, dst_addr, op0_addr, op1_addr)
 E.  mem_v     (4)  : Memory values (inst, dst, op0, op1)
 F.  offsets   (3)  : (off_dst, off_op0, off_op1)
 G.  derived   (3)  : (t0, t1, mul)

 A                B C  D    E    F   G
|xxxxxxxxxxxxxxxx|x|xx|xxxx|xxxx|xxx|xxx|

  1. Let rmin​ and rmax​ be respectively the minimum and maximum values of the entries of the submatrix Moffsets​ defined by the columns of the group offsets. Let v be the vector of all the values between rmin​ and rmax​ that are not in Moffsets​. If the length of v is not a multiple of three, extend it to the nearest multiple of three using one arbitrary value of v.

  2. Let R be the last row of M, and let R′ be the vector that's equal to R except that it has zeroes in entries corresponding to the ordered set of columns mem_a and mem_v. The set is ordered incrementally by mem_a. Let Lpub​ be the length of the public input (program code). Extend M with additional L′:=⌈Lpub​/4⌉ rows to obtain a matrix M∈F(L+L′)×33 by appending copies of R′ at the bottom (the notation ⌈x⌉ means the ceiling function, defined as the smallest integer that is not smaller than x).

  3. Let R′′ be the vector that's equal to R except that it has zeroes in entries corresponding to the set of columns mem_a and mem_v, let Maddr​ be the submatrix defined by the columns of the group addresses, let L′′=(L0′′​,L1′′​,...,LJ′′​)T the submatrix that asserts Maddr,i,j​<L0,j′′​, LI,j′′​<Maddr,i+1,j​ where Maddr,i+1,j​−Maddr,i,j​>1 and 0≤j≤J and I=∣Lj′′​∣. Extend M with additional L′′ rows to obtain a matrix M∈F(L+L′+L′′)×33 by appending copies of R′′ at the bottom.

  4. Pad M with copies of its last row until it has a power of two number of rows. As a result we obtain a matrix T∈F2n×33.
